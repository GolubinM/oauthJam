
### Запуск по триггеру скрипта statKeyWords.
Для запуска по триггеру скрипта statKeyWords необходимо чтобы в клиентском файле присктствовала функция вызова по тригеру statKeyWordsTrigg()

### Запуск по триггеру скрипта analyticNmIdPeriodTrigg.
Для запуска по триггеру скрипта statKeyWords необходимо чтобы в клиентском файле присктствовала функция вызова по тригеру analyticNmIdPeriodTrigg()

Вск скрипты которые содержа в параметрах (SP) допускают установку триггеров и продолжение сбора информации используя параметры скипта Property Service.

Задача:

Проверка доступа по учетной записи(gmail аккаунт):
все триггеры устанавливаются от имени пользователя.
При инициализации системы запуск скрипта установки триггеров через меню.
Из библиотеки выполняем скрипт проверки аккаунта, вызовом функции WebApp.
